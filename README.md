# Critix

Projeto de Conclusão de Curso (TCC) desenvolvido como parte do Ensino Médio Técnico em Tecnologia da Informação.

O Critix consiste em um sistema desenvolvido com foco na aplicação prática de conceitos de programação, lógica, estrutura de dados e organização de software, atendendo aos requisitos propostos no projeto final do curso.

---

## Objetivo do Projeto

Desenvolver um sistema funcional que aplicasse, na prática, os conhecimentos adquiridos durante o curso técnico, incluindo:
- Levantamento de requisitos
- Definição de funcionalidades
- Implementação da lógica de programação
- Estruturação de dados
- Testes e validação do sistema

---

## Funcionalidades

- Implementação das regras de negócio definidas no projeto
- Processamento e validação de dados
- Estrutura organizada do sistema
- Fluxo de uso conforme requisitos levantados
- Apresentação final do sistema como avaliação de conclusão de curso

---

## Tecnologias Utilizadas

- Linguagem de Programação: HTML, CSS, JavaScript, PHP
- Paradigma: Programação Estruturada / Orientada a Objetos
- Banco de Dados: Utilização de API do site TMDB e API do Google Books
- Ferramentas de Desenvolvimento: VSCode

> Obs: As tecnologias podem variar conforme a versão e escopo do projeto desenvolvido durante o curso.

---

## Contexto Acadêmico

- Projeto desenvolvido em grupo
- Parte obrigatória para conclusão do Ensino Médio Técnico em TI
- Envolveu trabalho em equipe, organização de tarefas e apresentação formal
- Avaliado por banca/professores da instituição

---

## Aprendizados

Durante o desenvolvimento do Critix, foi possível desenvolver e aprimorar habilidades como:
- Lógica de programação
- Organização de código
- Trabalho em equipe
- Resolução de problemas
- Comunicação técnica e apresentação de projeto
